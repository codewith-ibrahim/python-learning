## 📂 Repository Structure & Topics Covered

Ye poori repository structural folders mein divided hai taake aap har topic ko step-by-step seekh sakein:

### 📁 1. Basics & Setup
*   `1.print.py` — Screen par output dikhane ka aasan tareeqa.
*   `2.comments.py` — Code mein notes/comments likhna.
*   `3.variables.py` — Data store karne ke containers.
*   `4.pip_and_modules.txt` — External packages aur libraries install karna.

### 📁 2. Core Fundamentals
*   `1.operators.py` — Arithmetic, comparison aur logical operations.
*   `2.escape_characters.py` — Special formatting characters (e.g., `\n`, `\t`).
*   `3.string_indepth.py` — Strings ko detail mein handle karna.
*   `4.string_formatting.py` — Dynamic text representation (f-strings).
*   `5.user_input.py` — Console ke zariye user se data lena.

### 📁 3. Control Flow & Structure
*   `1.if_else.py` — Decision-making logic.
*   `2.if_else_practice/` — Conditionals ki practice exercises.
*   `3.match_case.py` — Pattern matching (modern switch-case).
*   `4.match_case_practice/` — Match-case ke real-world examples.
*   `5.for_loop.py` — Iterations aur lists/sequences par loop chalana.
*   `6.while_loop.py` — Condition-based loops.
*   `7.functions.py` — Reusable code blocks banana.
*   `8.functions_practice/` — Functions ki coding exercises.
*   `9.try_except.py` — Errors aur exceptions ko crash-free handle karna.
*   `10.indentation.py` — Python ke blocks aur spacing rules.
*   `11.file_handling.py` — Files ko read, write aur append karne ka tareeqa.

### 📁 4. Data Structures
*   `1.list_and_methods.py` — Ordered collections aur unke built-in methods.
*   `2.tuples_and_methods.py` — Immutable collections (jo change nahi ho sakein).
*   `3.sets_and_methods.py` — Unique elements ka collection.
*   `4.sets_theory.txt` — Sets ki theoretical concepts.
*   `5.dictionary_and_methods.py` — Key-Value pairs ka data store karna.

### 📁 5. Object-Oriented Programming (OOP)
*   `1.classes_and_objects.py` — OOP ke basic building blocks (Classes & Objects).
*   `2.constructors_and_self.py` — `__init__` constructor aur `self` keyword ka use.
*   `3.inheritance.py` — Code reusability aur parent-child classes.
*   `4.polymorphism.py` — Method overriding aur dynamic behavior.
*   `5.encapsulation_and_abstraction.py` — Data hiding aur secure access modifiers.

### 📁 6. Documentation
*   `common_confusions.txt` — Python seekhte waqt aane wali common mistakes aur unke solutions.
*   `env_file_explained.txt` — Environment variables aur sensitive data ko secure rakhne ka tareeqa.
*   `virtual_environment_explained.txt` — Virtual environments (`venv`) ki zaroorat aur settings.
*   `requirements_explained.txt` — Dependencies/libraries ko track karne ke liye requirements file ka use.
*   `gitignore_explained.txt` — Git repository se unnecessary files ko ignore karne ka tarika.

### 📁 7. Projects

#### 🌤️ 1. Weather App (`projects/1.weather_app/`)
*   `main.py` — Weather app ka main functional script.
*   `project_explained.txt` — Is project ki line-by-line detailed explanation.
*   `.env` — Environment variables config (API key).
*   `.gitignore` — Ignore files settings.
*   `requirements.txt` — App dependencies (requests, python-dotenv).

#### 📱 2. QR Code Generator (`projects/2.qr_code_generator/`)
*   `basic.py` — Simple QR Code (2 lines, no logo).
*   `generator.py` — QR generation logic (Auto-detect logo, favicon download).
*   `cli.py` — Command Line Interface (User se interaction).
*   `logos/` — Logo images folder (GitHub, LinkedIn, YouTube, etc.).
*   `qr_code_explained.txt` — Is project ki line-by-line detailed explanation.
*   `.gitignore` — Ignore files settings.
*   `requirements.txt` — App dependencies (qrcode, pillow, requests).
