## ЁЯУВ Repository Structure & Topics Covered

Ye poori repository structural folders mein divided hai taake aap har topic ko step-by-step seekh sakein:

### ЁЯУБ 1. Basics & Setup (`basics_and_setup/`)
*   `1.print.py` тАФ Screen par output dikhane ka aasan tareeqa.
*   `2.comments.py` тАФ Code mein notes/comments likhna.
*   `3.variables.py` тАФ Data store karne ke containers.
*   `4.pip_and_modules.txt` тАФ External packages aur libraries install karna.

### ЁЯУБ 2. Core Fundamentals (`core_fundamentals/`)
*   `1.operators.py` тАФ Arithmetic, comparison aur logical operations.
*   `2.escape_characters.py` тАФ Special formatting characters (e.g., `\n`, `\t`).
*   `3.string_indepth.py` тАФ Strings ko detail mein handle karna.
*   `4.string_formatting.py` тАФ Dynamic text representation (f-strings).
*   `5.user_input.py` тАФ Console ke zariye user se data lena.

### ЁЯУБ 3. Control Flow & Structure (`control_flow_and_structure/`)
*   `1.if_else.py` тАФ Decision-making logic.
*   `2.if_else_practice/` тАФ Conditionals ki practice exercises.
*   `3.match_case.py` тАФ Pattern matching (modern switch-case).
*   `4.match_case_practice/` тАФ Match-case ke real-world examples.
*   `5.for_loop.py` тАФ Iterations aur lists/sequences par loop chalana.
*   `6.while_loop.py` тАФ Condition-based loops.
*   `7.functions.py` тАФ Reusable code blocks banana.
*   `8.functions_practice/` тАФ Functions ki coding exercises.
*   `9.try_except.py` тАФ Errors aur exceptions ko crash-free handle karna.
*   `10.indentation.py` тАФ Python ke blocks aur spacing rules.

### ЁЯУБ 4. Data Structures (`data_structures/`)
*   `1.list_and_methods.py` тАФ Ordered collections aur unke built-in methods.
*   `2.tuples_and_methods.py` тАФ Immutable collections (jo change nahi ho sakein).
*   `3.sets_and_methods.py` тАФ Unique elements ka collection.
*   `4.sets_theory.txt` тАФ Sets ki theoretical concepts.
*   `5.dictionary_and_methods.py` тАФ Key-Value pairs ka data store karna.

### ЁЯУБ 5. Documentation (`documentation/`)
*   `common_confusions.txt` тАФ Python seekhte waqt aane wali common mistakes aur unke solutions.
*   `README.md` тАФ Is repository ka guide map.
